\## 1. 今天学了什么
-1. Simple functions，分为两部分，header和body
-Header（函数头）定义名称和argument（参数）
-每个函数头都以 def 开始，这告诉 Python 我们即将定义一个函数。
-argument 是将被用作函数输入的变量的名称，它总是被括在函数名称之后立即出现的括号中。（but函数也可以没有参数，或者可以			  有多个参数。
-对于每个函数，括在函数参数后面的括号必须以“：”结尾。
-Body（函数体）指定函数所做的工作。
-函数体中的每一行代码都必须精确缩进四个空格——按下空格键四次or按一次“Tab”键。
-返回语句，只是某参数的值作为函数的输出返回。
-2. Run/call 运行函数
-new\_number = 函数名（输入值）
-eg. 		#Define the function
		def add\_three(input\_var):
		output\_var = input\_var + 3
		return output\_var



&#x09;	#Run the function with 10 as input

&#x09;	new\_number = add\_three(10)  //Check that the value is 13, as expected

&#x09;	print(new\_number)

\-3. Name 命名，函数命名时，只使用小写字母，用下划线而不是空格分隔单词。
