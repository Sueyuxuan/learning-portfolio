\## 1. 今天学了什么

\- \*\*Simple Functions（简单函数）\*\*

&#x20; - 函数由两部分组成：Header（函数头）和 Body（函数体）。

&#x20; - Header（函数头）用于定义函数名称和 argument（参数）。

&#x20; - 每个函数头都以 `def` 开始，这告诉 Python 我们即将定义一个函数。

&#x20; - argument（参数）是函数输入变量的名称，写在函数名后的括号 `()` 中。

&#x20; - 函数可以没有参数，也可以有一个或多个参数。

&#x20; - 每个函数头最后都必须以冒号 `:` 结尾。

&#x20; - Body（函数体）用于指定函数执行的内容。

&#x20; - 函数体中的每一行代码都必须缩进 4 个空格（按四次空格键或一次 `Tab`）。

&#x20; - 使用 `return` 语句将结果作为函数的输出返回。

\- \*\*Run / Call（运行函数）\*\*

&#x20; - 调用函数的格式：

```python

new\_number = 函数名(输入值)

```

&#x20; - 示例：

```python

\# Define the function

def add\_three(input\_var):

&#x20;   output\_var = input\_var + 3

&#x20;   return output\_var

\# Run the function with 10 as input

new\_number = add\_three(10)

\# Check that the value is 13, as expected

print(new\_number)

```

\- \*\*Naming（函数命名）\*\*

&#x20; - 函数名通常只使用小写字母。

&#x20; - 多个单词之间使用下划线 `\_` 连接，而不是空格。
